## 📌 About the Project

A **College Lost and Found Website** that provides a secure and easy-to-use platform for the campus community. Students and staff can report lost items, post found items, and connect with each other to return belongings — all within a verified college environment.

---

## ✨ Features

- 🔐 **Phone-based Registration & Login** — Indian numbers only (+91), OTP verified
- 📋 **Report Lost or Found Items** — with photo, category, location, description
- 📞 **Public Contact Details** — Indian phone number visible on every post
- 🗓️ **Auto 30-Day Expiry** — posts automatically removed after 30 days
- 🔎 **Search & Filter** — by category, location, Lost/Found status
- 📬 **Smart Claim System** — submit proof of ownership to claim an item
- ⚙️ **Admin Panel** — manage all posts and claims
- 📱 **Fully Responsive** — works on mobile and desktop
